## 🏛️ Visão Geral

Este repositório apresenta a construção progressiva de uma **Landing Page Institucional de Advocacia**, desenvolvida com React de forma modular, escalável e seguindo padrões profissionais de mercado.

Mais do que criar uma interface, o projeto ensina:

- Arquitetura de componentes  
- Organização escalável de arquivos  
- Uso correto de estado  
- Boas práticas de versionamento com Git  
- Mentalidade de desenvolvimento orientado a produto  

> ⚠️ **Este não é um curso de React.**  
> É um curso sobre **como pensar como um desenvolvedor profissional**.

---

## 🎯 Objetivo do Projeto

Desenvolver uma landing page institucional completa contendo:

- ✅ Header profissional  
- ✅ Hero Section estratégica  
- ✅ Seção "Sobre o Escritório"  
- ✅ Áreas de Atuação  
- ✅ Depoimentos dinâmicos com `map()`  
- ✅ Formulário com validação via `useState`  
- ✅ Footer institucional com ano automático  
- ✅ Arquitetura organizada e escalável
